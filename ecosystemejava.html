Description

Le langage Java est un langage vaste qui permet de nombreuses possibilités.
Toutefois, les 3 principaux sous-ensembles de Java sont :
- Java SE qui permet de créer des applications client-lourd
- Java EE qui permet de créer des applications web
- Android qui permet de créer des applications mobiles
La version actuelle de Java est la version 8.

Java SE


Destiné essentiellement aux applications de bureau, Java SE dispose de son propre framework graphique, qui permet de créer des interfaces.
Jusqu'a la version 7, le framework graphique de JSE est Swing.
La version 8 de Java introduit un nouveau framework graphique, Java FX, qui se rapproche plus du material design dans ses interfaces.

Java EE


JEE est destiné aux applications web, en local ou sur le web, mais aussi aux sites internet.
Il est particulièrement intéressant pour l'utilisation des webservices, qui sont, comme leur nom l'indique, des services disponibles sur le web.
Le plus souvent ce sont des fonctionnalités que l'on peux intégrer à un site web.

Android


Android à pour base le langage Java, mais dispose de ses propres fonctionnalités.
