Outil de gestion de version

Un logiciel de gestion de versions est un logiciel qui permet de stocker un ensemble de fichiers en conservant la chronologie de toutes les modifications qui ont été effectuées dessus.

La gestion de source s'articule donc autour de 3 éléments:
  • Historisation
  • Partage
  • Organisation
Git quant à lui est un logiciel de gestion de source mais décentralisé. Il consiste à voir l'outil de gestion de versions comme un outil permettant à chacun de travailler à son rythme, de façon désynchronisée des autres, puis d'offrir un moyen à ces développeurs de s'échanger leur travaux respectifs.
